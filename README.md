# 三角洲小助手插件

一个基于真寻机器人的三角洲信息查询插件，可以获取三角洲游戏中的地图密码和制作产物利润信息。

## 功能特性

- 获取三角洲各大地图密码
- 查询制作产物的当前利润
- 支持群聊和私聊转发消息
- 自动处理Cookie和版本信息

## 安装方法

1. 将本插件文件夹放置到真寻机器人的 `plugins` 目录下
2. 确保已安装必要的依赖：
   - nonebot-plugin-alconna
   - httpx

## 使用说明

### 指令格式

```
洲 或 粥
```

### 返回信息

插件会返回以下信息：

1. **地图密码信息**：

   - 零号大坝密码
   - 长弓溪谷密码
   - 巴克什密码
   - 航天基地密码
   - 潮汐监狱密码
2. **制作产物利润信息**：

   - 技术中心产物及利润
   - 工作台产物及利润
   - 制药台产物及利润
   - 防具台产物及利润
